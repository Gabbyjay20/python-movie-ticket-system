# Python Movie Ticket Booking System

## Overview
This project is a simple Python application that allows users to view available movies, book tickets, and see their booking summary. The program demonstrates basic programming concepts such as variables, loops, conditionals, functions, and data structures.

## Purpose
The purpose of this project is to practice Python programming and demonstrate the ability to build a small software application that interacts with users and processes data.

## Features
- Displays a list of available movies
- Allows users to select a movie
- Calculates the total ticket price
- Applies a discount for group bookings
- Stores bookings in a list
- Displays a booking summary

## Technologies Used
- Python 3
- Command Line Interface (CLI)

## How to Run the Program

1. Make sure Python is installed on your computer.
2. Download or clone this repository.
3. Navigate to the project folder.
4. Run the program using the following command:

python main.py

## Example Menu

====== Movie Ticket System ======

1. View Movies  
2. Book Ticket  
3. View My Bookings  
4. Exit  

Users can choose an option and follow the prompts to book tickets.

## Code Structure
The program is organized into several functions:

- `show_movies()` – displays available movies  
- `calculate_total()` – calculates the ticket price  
- `book_ticket()` – handles the booking process  
- `view_bookings()` – shows booking history  
- `show_menu()` – displays the main menu  
- `main()` – runs the program loop
