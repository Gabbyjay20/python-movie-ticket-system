# Python Movie Ticket System

A simple command-line movie ticket booking system written in Python.

## Features

- View available movies with prices
- Book movie tickets
- Group booking discount (10% off for 5+ tickets)
- View all bookings

## Movies Available

| # | Movie | Price |
|---|-------|-------|
| 1 | Avengers | $10 |
| 2 | Black Panther | $12 |
| 3 | Spider-Man | $11 |
| 4 | The Batman | $13 |

## How to Run

```bash
python main.py
```

## Usage

1. Select option 1 to view movies
2. Select option 2 to book a ticket
3. Select option 3 to view all bookings
4. Select option 4 to exit

## Discount Policy

- 10% discount applied automatically for group bookings of 5 or more tickets

## Requirements

- Python 3.x
